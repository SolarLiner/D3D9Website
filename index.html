Main contributors

Jarmonik is the founder of the project. He built the engine from scratch and should be the one receiving all the glory.

Kuddel is what some would describe as the number 2 on the project, doing all sorts of superhero work to make things happen.

Bibi Uncle made runways visible at night by providing working code for PAPI lights and varioud landing indicators.

SolarLiner runs the experimental HDR version of the graphics client, made for the more advanced and recent graphics card. (Am I a main contributor yet? :P)
